# kur

Bu bir build komutu. Asagidaki adimlari takip et:

1. Bu dizindeki `CLAUDE.md` dosyasini oku — uygulama spesifikasyonunun tamami orada.
2. Spesifikasyona birebir uygun sekilde `index.html` dosyasini olustur.
3. Tek dosya: inline CSS + inline JS, harici dosya yok.
4. Print CSS'in dogru calistigini dogrula — sadece fatura onizlemesi yazdirilmali.
5. On yuklu ornek faturanin hesaplamalari dogru olmali (KDV, toplam).
6. Tamamlaninca tarayicida acilmaya hazir oldugunu bildir.
